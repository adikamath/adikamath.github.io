# Aditya Kamath

Personal blog and notes on machine learning, GenAI, developer tools, products, and things I'm learning along the way.

Live site: [adityakamath.com](https://adityakamath.com)

Built with Jekyll and maintained as a lightweight personal publishing setup.

## Local Development

```bash
bundle install
bundle exec jekyll serve
```

Open `http://127.0.0.1:4000`.
